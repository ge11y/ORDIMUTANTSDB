# OrdiMutants Archive Vision

This project is a community site and explorer for the OrdiMutants ordinal collection on Bitcoin.

The goal is to create a place where collectors and community members can:

- Browse the entire OrdiMutants collection
- View rare satributes and traits
- View the slug associated with each mutant
- Track mutants they own or are watching
- Discover interesting mutants through featured highlights

The site should feel like an ancient archive or registry rather than a modern NFT website.

Visual inspiration:
- rustic
- ancient
- nordic / viking inspiration
- runic motifs
- weathered textures
- historic archive feeling

The OrdiMutants banner included in the repository will be used as the visual reference for the site's typography and branding.

The site will also host future community tools and projects related to the OrdiMutants ecosystem.
