# Core Features

## Public Features

- Gallery of all OrdiMutants
- Search by name, inscription id, or number
- Trait and satribute filtering
- Mutant detail page
- Rare sat badges
- Slug viewer for each mutant

## User Accounts

Users can create accounts without wallet login.

Users can track mutants with the following statuses:

- Favorite
- Watching
- Owned
- Wishlist

Users will have a dashboard showing mutants grouped by these statuses.

## Featured Mutant System

The site will highlight a "Featured Mutant of the Week".

A page called "Hall of Featured" will archive past featured mutants.

## Social Signals

Mutant pages will show how many collectors are watching a mutant.

Example:
"12 collectors watching this mutant"

## Trait Explorer

Users can search for mutants with specific trait combinations or satributes.
