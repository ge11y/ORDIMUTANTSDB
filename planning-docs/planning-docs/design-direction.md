# Visual Direction

The site should feel like an ancient archive or registry.

Inspiration:
- nordic / viking design
- runic symbols
- carved stone
- weathered parchment
- ancient manuscripts

Avoid modern NFT aesthetics such as neon, cyberpunk, or glossy tech styles.

The OrdiMutants banner and logo should be used as the reference for typography and branding.

Suggested visual elements:

- dark wood textures
- stone and iron accents
- parchment colored panels
- muted gold highlights
- runic divider elements

The goal is to make the site feel historic, rustic, and timeless.
