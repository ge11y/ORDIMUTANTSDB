# OrdiMutants HQ - Standalone File

## Files to Download
1. `ORDIMUTANTHQ-v8-standalone-ready.html` - The main HTML file

## Important Note
This HTML file references local assets in the `assets/` folder:
- Logo images (`assets/branding/`)
- Badge images (`assets/badges/`)

**To view the file properly**, you need:
- The `assets` folder with all images
- Both files in the same folder

## Alternative
If you want a truly standalone file (no assets folder needed), you would need to host the assets online and update the URLs, or convert images to base64 data URLs.
